Решение для укладки в короб
Кобот с помощью специального захвата берет единичную упаковку с конвейера и укладывает в групповую тару, следуя строго по заданному алгоритму
Заполненная тара перемещается дальше по конвейеру, а в зону работы кобота подъезжает следующая пустая тара
Существует возможность программирования укладок по различным схемам, выбор рецепта осуществляется через панель оператора
Установка имеет компактные размеры, что позволяет легко интегрировать в существующую производственную линию
По желанию заказчика кобот-укладчик может быть дополнительно оснащен оборудованием: конвейером и магазином прослоек
