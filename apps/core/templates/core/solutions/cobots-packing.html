Решение для паллетизации промышленным роботом
Он поддерживает более широкий формат укладываемой продукции, включая короба, мешки и емкости
Промышленный робот с помощью специального захвата берет изделия с конвейера и автоматически выполняет автоматичекое паллетирование, формируя паллеты по заданному алгоритму. Захват разрабатывается индивидуально для каждого формата продукции
Укладка может осуществляться как с одной стороны, так и поочередно с двух сторон. Замену наполненных паллет может выполняться оператором или с использованием специального конвейера
Существует возможность программирования укладок по различным схемам для различных видов упаковки, а также выбора рецепта через панель оператора
По желанию заказчика робот-паллетайзер может быть дополнительно оснащен оборудованием: конвейером для подачи продукции, магазинном для паллет, конвейером для автоматичесой подачи паллет и вывода из зоны укладки наполненных паллет, накопительным столом, магазином гофролистов и защитным ограждением
