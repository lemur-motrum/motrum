Решение для паллетизации
Коллаборативный робот (кобот) с помощью специального захвата берет изделия с конвейера и автоматически укладывает их на поддон, формируя паллет по заданному алгоритму
Укладка осуществляется поочередно: кобот заполняет один паллет, затем переходит к следующему. Оператор заменяет заполненный паллет на пустой, что обеспечивает непрерывный режим работы кобота
Существует возможность программирования укладок по различным схемам для различных видов упаковки, а также выбора рецепта через панель оператора
Установка имеет компактные размеры, что позволяет легко интегрировать в существующую производственную линию
По желанию заказчика кобот-паллетайзер может быть дополнительно оснащен оборудованием: конвейером, накопительным столом, магазином гофролистов и защитным ограждением
